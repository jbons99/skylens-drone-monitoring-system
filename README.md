# SkyLens – Drone Monitoring Platform

SkyLens is a full-stack web application that uses drone-captured aerial imagery to detect and track people and vehicles in real time.

The platform allows users to upload images, process them using a YOLO-based object detection model, and visualise results through an interactive dashboard with analytics and historical data.

This project was developed as part of a university team assignment and demonstrates full-stack development, machine learning integration, and real-time data processing.

## My Contributions

- Developed the frontend interface using React and Tailwind CSS  
- Built key UI components including dashboard, authentication pages, and upload system  
- Implemented data visualisation using Recharts  
- Integrated frontend with backend APIs
